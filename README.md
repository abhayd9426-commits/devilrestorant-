# DEVIL Restaurant App

A complete full-stack simulated restaurant application built with React, Tailwind CSS, and a Mock LocalStorage Database.

## Features
- **User Authentication**: Sign up and Login.
- **Menu System**: Browse categories, view items, and manage a shopping cart.
- **Admin Panel**: Dashboard stats, Manage Menu (Add/Edit/Delete), and Order Management (Confirm/Reject/Deliver).
- **Order Tracking**: Real-time status updates for users.
- **Responsive Design**: Works on Mobile, Tablet, and Desktop.

## How to Save to GitHub

Since this code is generated in an AI sandbox, you must move it to your local machine to save it to GitHub.

### Step 1: Download/Copy Code
Create a folder on your computer (e.g., `devil-restaurant`) and save all the provided files into it, maintaining the folder structure (e.g., `src/pages`, `src/components`, etc.).

### Step 2: Initialize Git
Open your terminal/command prompt in that folder and run:

```bash
git init
git add .
git commit -m "Initial commit of DEVIL Restaurant App"
```

### Step 3: Connect to GitHub
1. Go to [GitHub.com](https://github.com) and create a new repository (e.g., `devil-restaurant`).
2. Copy the repository URL (e.g., `https://github.com/your-username/devil-restaurant.git`).
3. Run the following commands in your terminal:

```bash
git branch -M main
git remote add origin https://github.com/your-username/devil-restaurant.git
git push -u origin main
```

## How to Run Locally

Prerequisites: Node.js installed.

1. **Install Dependencies**:
   ```bash
   npm install react react-dom react-router-dom lucide-react
   npm install -D tailwindcss postcss autoprefixer
   ```

2. **Start the App**:
   ```bash
   npm start
   ```
